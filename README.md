# East Tennessee Hub (myTN)

An agentic program that routes user intentions to category-specific agents and stores neat local action receipts.

## Categories
- TA: IT Agent
- SA: Sales Agent
- PA: Press Agent
- FA: Farming Agent
- WA: Wildlife Agent
- CA: Church Agent
- JA: Job Agent
- ArA: Arts Agent
- RA: Recreation Agent

## Run
```bash
python main.py
```

Commands:
- `run`  → select an intention & goal
- `list` → view recent receipts
- `quit` → exit

## Requirements
- Python 3.10+
